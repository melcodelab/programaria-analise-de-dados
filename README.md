Repositório de aprendizado com conteúdo, exercícios e projetos do curso **Introdução a Análise de Dados com Python**, oferecido pela [PrograMaria](https://www.programaria.org/), iniciativa brasileira dedicada a aumentar a presença de mulheres na tecnologia.

---

## O que é PrograMaria?

A PrograMaria é uma iniciativa brasileira que busca aumentar a presença de mulheres na tecnologia. Nasceu da percepção de que o gap de gênero em programação não vem de falta de capacidade, mas de barreiras sistêmicas: falta de modelos inspiradores, estereótipos persistentes e ambientes muitas vezes hostis.

A organização trabalha para empoderar meninas e mulheres através de cursos acessíveis, conteúdos inspiradores e eventos de networking, mostrando que elas são capazes de realizar suas próprias ideias.

## Sobre este Repositório

Aqui documento meu aprendizado em análise de dados, desde os primeiros passos com Google Sheets até Machine Learning com Python. Você vai encontrar:

- Anotações dos módulos do curso
- Análises exploratórias sobre o mercado de tecnologia brasileiro
- Jupyter Notebooks com código python executável
- Consultas SQL e modelos de dados
- Gráficos e visualizações geradas ao longo do curso

---

## 📚 Estrutura das anotações

| Módulo | Tema | Foco |
|--------|------|------|
| **1** | Análise exporatória com Google Sheets | Tabelas dinâmicas, estatística básica, filtros e  criação de gráficos |
| **2–3** | Python & Pandas | Limpeza, manipulação e análise de dados |
| **4** | Banco de Dados & SQL | Modelagem, consultas, integração com Python (SQLite3) |
| **5** | Visualização de Dados | Matplotlib, Google Data studio, Seaborn, Plotly e storytelling |
| **6** | Regressão Linear | Previsões, pré-processamento de dados e aprendizado supervisionado |

### Perguntas que exploramos com os dados

- Quanto maior a formação, maior o nível profissional?
- Tem mais homens ou mulheres com Pós-Graduação no setor de tech?
- Qual a faixa de idade mais comum entre profissionais de dados?
- Quais setores concentram mais profissionais de tecnologia?
- Quais fatores influenciam mais o salário?
- É possível prever o salário com Machine Learning?
